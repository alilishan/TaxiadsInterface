# Taxi Ads
 Interface Design for TaxiAds User BackEnd

* bootstrap-3.3.7
* font-awesome-4.6.3
* [amcharts](https://www.amcharts.com)

* [checkall - Works with jquery-2.2.4 and below check js file](https://github.com/mjball/jQuery-CheckAll)
